# Changelog

All notable changes to the Gathering Architect skill are recorded here. This skill is licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).

## 3.0.0

**Stronger coaching, more trustworthy evidence.**

- The skill now holds its "understand before you build" approach even when you're in a hurry. It acknowledges your deadline and promises the deliverable, then still asks the one or two questions that genuinely change the shape of your event.
- Diagnostic questions are sharper and tied to a real decision, so the first reply feels specific rather than generic.
- Added an honest gut-check instinct: a good coach will gently name the uncomfortable possibility, including whether a gathering has run its course or shouldn't happen at all.
- Cleaned up the research it cites. Removed widely-circulated but discredited or unsourced statistics (the "learning pyramid" retention percentages, blog-sourced cohort-completion and motivation figures, and over-precise effect sizes with no traceable study). The skill now relies only on named, peer-reviewed sources such as Freeman et al. (2014, PNAS) and Edmondson (1999), and includes a short list of figures it deliberately avoids so they don't creep back in.

## 2.0.0

**First public release.**

- Trimmed and reorganized the reference library so each topic has one clear home (for example, Open Space Technology, Communities of Practice, and the Minimum Viable Community tool each live in a single place). Roughly 15% smaller overall, with no loss of substance.
- Removed leftover draft notes from the reference files.
- Added five ready-to-use workshop templates: a Minimum Viable Community worksheet, an IDOARRT meeting-design tool, a run-of-show agenda with energy notes, a Community Canvas facilitator guide, and an agenda-builder prompt.
- Added an index to the reference files and tightened how the skill decides which one to open.
